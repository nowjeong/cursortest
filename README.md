## 📖 게임 소개

구구단 산성비 게임은 하늘에서 떨어지는 구구단 문제를 빠르게 풀어 점수를 얻는 교육용 게임입니다. 재미있는 게임을 통해 자연스럽게 구구단을 학습하고 암기할 수 있습니다.

### ✨ 주요 기능

- 🎮 **5단계 난이도**: 입문부터 달인까지, 실력에 맞는 난이도 선택
- 📊 **점수 시스템**: 정답마다 점수 획득, 콤보 보너스로 추가 점수
- ❤️ **생명력 시스템**: 3개의 생명력으로 긴장감 있는 게임
- 🏆 **최고 점수 저장**: 난이도별 최고 점수 기록
- 📱 **반응형 디자인**: PC, 태블릿, 모바일 모두 지원
- 🎨 **친근한 UI/UX**: 아이들이 쉽게 이해할 수 있는 직관적인 디자인

## 🎯 난이도별 특징

| 난이도 | 구구단 범위 | 속도 | 동시 문제 수 | 시간 제한 |
|--------|-------------|------|--------------|-----------|
| 🟢 입문 | 2단, 5단 | 느림 | 1-2개 | 90초 |
| 🔵 초급 | 2-5단 | 느림 | 2-3개 | 90초 |
| 🟡 중급 | 2-9단 | 보통 | 2-3개 | 60초 |
| 🟠 고급 | 2-9단 | 빠름 | 3-4개 | 60초 |
| 🔴 달인 | 2-9단 | 매우 빠름 | 4-5개 | 45초 |

## 🚀 시작하기

### 사전 요구사항

- Node.js 18.0.0 이상
- npm 또는 yarn

### 설치 방법

1. 저장소 클론 또는 파일 다운로드

2. 의존성 패키지 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```

4. 브라우저에서 `http://localhost:5173` 접속

### 빌드

프로덕션 빌드 생성:
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

빌드 미리보기:
```bash
npm run preview
```

## 🎮 게임 방법

1. **난이도 선택**: 메인 화면에서 자신의 실력에 맞는 난이도를 선택하세요.

2. **게임 시작**: "게임 시작!" 버튼을 클릭하면 게임이 시작됩니다.

3. **문제 풀기**: 
   - 화면 위에서 구구단 문제가 빗방울처럼 떨어집니다
   - 화면 하단의 답 버튼을 클릭하여 정답을 맞추세요
   - 정답을 맞추면 빗방울이 사라지고 점수를 얻습니다

4. **생명력 관리**:
   - 시작 시 3개의 생명력(❤️)이 주어집니다
   - 오답을 선택하거나 문제가 바닥에 닿으면 생명력이 1개씩 감소합니다
   - 모든 생명력을 잃으면 게임이 종료됩니다

5. **보너스 점수**:
   - 기본 점수: 정답 1개당 10점
   - 빠른 정답: +5점 보너스
   - 5연속 정답: +50점 보너스
   - 10연속 정답: +100점 보너스

## 🔧 개발자 정보

### 커스터마이징

난이도 조정이나 점수 시스템을 수정하려면 `src/utils.ts` 파일의 다음 상수들을 변경하세요:

- `DIFFICULTY_CONFIGS`: 난이도별 설정
- `GAME_CONSTANTS`: 게임 기본 상수

### 스타일 수정

`src/styles.css` 파일에서 색상, 크기, 애니메이션 등을 수정할 수 있습니다.
